pulse.Engine(params)
Engine handles starting the game, generating the game loop and handling the base game window
Author: PFP.
Parameters:
{object} params
parameters that can be set as initialized options on the node
{string|DOMElement} params.gameWindow
the dom element to use as the main game window, if a string it will pull the element by id, if it's a element it will simply use it as is
{size} params.size Optional
initial size width and height to use for the engine's game window.
